# snhu-cs-330-comp-graphics-and-visualization
Portfolio for my SNHU course

This was really the first time I've done any computer graphics work. I enjoyed learning a bit about OpenGL and being able to spend some more time in C++. I generally like to build higher-level solutions, but learning more about lower level programming and understanding the math involved in 3D graphics, especially being able to utilize some of the linear algebra, was very interesting.

I find that breaking large problems into smaller goals helps me to be able to tackle large daunting problems. Sometimes if there is a part of a problem that isn't clear to me I will start with the parts that make sense and go back to the more difficult issues later. This usually helps me understand how the unclear parts of the problem relate to the rest. Iterating upon a solution allows me to make small changes as the solution becomes clearer and test small parts of a program.

I'm not sure how much computer graphics I will be using in the future, but this experience did give me another opportunity to practice iterating towards a solution. Each step of the process, from creating complex shapes to final lighting decisions, allowed me to implement some code changes and see how those changes affected my solution. As more complication was added to the scene I was able to go back and adsjust the structure of my code to end up with a cleaner and more readable project.
